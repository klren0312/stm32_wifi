>代码地址：https://github.com/klren0312/stm32_wifi

使用的器件：

 - STM32F103CBT6 单片机
 - ESP8266-01S wifi模块
 - SHT20温湿度
 - BT05蓝牙模块
 - 反射式红外

使用的Node.js服务端：

 - net模块
 - express
 - node-mysql
