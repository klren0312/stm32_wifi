使用的器件：

 - STM32F103CBT6 单片机
 - ESP8266-01S wifi模块
 - SHT20温湿度
 - BT05蓝牙模块
 - 反射式红外
 - 麦步手表
 - S3010舵机


使用的Node.js服务端：

 - net
 - express
 - mysql
 - socket.io
 - redis
